Editing an Animation#

When we use manim to create an animation, we send the output to a .mp4 file, which is then displayed in Jupyter Book. So, if we wanted to add a video in line, we can add the video to the correct file path, then write the following lines of code:

# from IPython.display import Video
# Video("filepath.mp4", embed=True, height=400)

An example of this can be found here

Changing an animation#

Now, the code for these animations are stored within the animations folder of the repository.

  1. To edit an animation, we begin by finding the corresponding code for the animation we are trying to edit. After editing the code for the animation, we then run the code that outputs a .mp4 file. Let us look at an example:

from IPython.display import Video
Video("./PotentialWell.mp4", embed=True, height=400)

  1. In the animations folder, the code for this is within the potential_well.py file.

  2. We can then navigate to this file in the code editor and make any change that we find necessary, then run the code. This will output the .mp4 file.

  3. The next step is to take this .mp4 file, and replace it with the one in the HRL Book that we are attempting to change.

  4. Finally, we need to make sure that the code that displays the .mp4 file contains the correct file path.
